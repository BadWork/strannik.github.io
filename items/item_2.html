Категория Палатки

60 000BYR(6 BYR )

Описание

Трехместная палатка Nishe 3 с двумя входами, что упрощает доступ к палатке.

Двухслойная  трехсезонная конструкция, которая предназначена для туристических походов в весеннее, летнее и осеннее время. В каркасе этой палатки используется верхняя консоль - это позволило увеличить объем тамбуров.Для вентиляции в палатке есть два вентиляционных клапана, которые регулируются по высоте стойкой.

Оба входа в палатку продублированы москитными сетками.

Для улучшения параметров водонепроницаемости все швы в тенте палатки проклеены специальной

лентой.

Особенности: 
- проклеенные швы;
- верхний слой палатки сделан из полиэстера 100% повышенной прочности;
- внутренняя палатка сделана из дышащего полиэстера;
- два входа и два тамбура;
- входы спальных отделений продублированы москитной сеткой;
- предусмотрены карманы для хранения различных мелочей;
- полочка для фонарика;
- отличная вентиляция;
- вентиляционные клапаны регулируемого размера;
- регулируемые оттяжки и зацепка для колышков;
- удобная конструкция входов;
- удобная сумка с ручками для переноски;
- в комплект входит ремнабор (кусочки ткани, запасная фурнитура) + одна запасная дуга

Модель: TRAMP Nishe 3
Размер: 220х320х120 см 
Вес: 3,76 кг 

Материал: 
- тент: 100% полиэстер 75D/190T RipStop WR PU 5000 мм вдс;
- внутр. тент: дышащий полиэстр 100%;
- дно: 100% полиэстер, 7000 мм вдс;
- дуги: фибергласс 8,5мм.
Тип: напряжённый каркас. 
Особенности: 2 входа
